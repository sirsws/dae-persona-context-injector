# DaE Agent-Store Copy Draft

## Title

**DaE: Persona Context Injector**

## Short Pitch

Most agents start working too early.
They plan before they understand the user.

DaE fixes that by generating a reusable `PersonaProfile` first.

## Core Message

DaE is not a chat gimmick.
It is an upstream context layer.

It forces concrete events, trade-offs, constraints, and tensions into a reusable profile that other agents can read before doing work.

## Why it matters

Without a profile, downstream agents drift toward generic answers.
With a profile, they can organize around the operator's actual priorities and decision logic.

## Best fit

Use DaE when you:

- work with multiple agents
- want a shared understanding layer
- need reusable context before planning or execution

## One-line close

**Profile first. Then collaborate.**
