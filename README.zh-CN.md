## 仓库角色

这个仓库继续作为 DaE skill 的公开分发与兼容入口。
它主要承载可安装的 skill 资产、基准材料和历史打包结构。

## 核心思路

DaE 围绕一个上游核心资产展开：`PersonaProfile`。
方法很简单，但后果很大：

**先建模用户，再让下游 agent 在更强上下文上工作。**

## 最适合的用途

如果你要直接使用公开的 DaE skill 包，用这个仓库。
如果你需要更完整的方法背景、分支地图和研究结构，进入 AKM 母港。
