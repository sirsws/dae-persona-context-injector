## 这个仓库现在是什么

这个仓库仍然保留为公开的 DaE skill 仓，但它不再被表述为与 AKM 平行的独立母概念。

它现在的角色更窄、更清楚：

- 场景：Persona / 顾问协作
- 上游方法：AKM
- 分支角色：首个完整参考实现
- 核心上游资产：`PersonaProfile`

## 为什么这很重要

DaE 真正解决的问题不是泛泛 prompt 技巧，而是上游建模问题：

**先把用户画像做对，再让下游 agent 开工。**
